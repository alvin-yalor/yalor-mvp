# **ACE React Demo**

A lightweight front-end demo for **ACE (AI Commerce Exchange)** showcasing the interaction between a user, a mock AI agent, and automated opportunity insights generated by ACE logic.  
This project is designed for rapid prototyping, demos, and future deployment on **AWS (EC2 or ECS Fargate)**.
